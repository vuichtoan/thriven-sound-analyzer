# Thriven Sound Analyzer – Spezifikation (kompakt)

## Kernidee
CLI scannt Audio-Ordner (Suno Exports), analysiert LUFS/Peak/RMS + Silence, rankt “brauchbare” Loops/Stems und exportiert Top-N.  
Zusatz: **8-Stem Ordnung** + **BR-864-ready Export**.

## BPM Range (Meta)
Standard 90–190 BPM (als Tag/Meta für Strukturierung; echte Detection optional später).

## 8 Stems (BR-864)
Slot 01..08, Slot 08 kann “Backing Vocals” sein.
